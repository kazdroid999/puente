よくある質問

FAQ

Micro SaaS オーナーになるのに技術知識は必要ですか？

いいえ。企画概要・対象ユーザー・欲しい機能をダッシュボードから投稿するだけです。AIが事業計画やBEPを自動設計し、Puenteが開発・運用・保守をすべて行います。

初期費用以外に費用はかかりますか？

月額プラットフォーム費は0円です。初期費用330,000円（税込）のみで、ホスティング・運用保守・障害対応すべて込みです。現在、先着の創業メンバー価格として80%OFFの66,000円で提供中です。

売上の分配はどうなりますか？

エンドユーザーの月額課金売上は、Stripe Connectにより自動で70（プエンテ）/ 30（オーナー）に分配されます。ロイヤリティ支払いではなく、各自の売上として計上されるため、インボイス対応も安心です。翌月末にオーナー指定口座へ自動振り込みされます。

エンドユーザー向けのサブスクプランは？

全SaaS共通で4プランあります。無料（月3回）/ ベーシック 月額980円（月30回）/ スタンダード 月額1,980円（月100回）/ プレミアム 月額2,980円（無制限）。すべて税込で、いつでもアップグレード・ダウングレード・解約可能です。

サブスクの解約やダウングレードはできますか？

はい。いつでもアプリ内のプラン画面から変更できます。ダウングレード・解約は次の請求期間から適用されます。現在の請求期間が終了するまでは引き続きご利用いただけます。

企画投稿からローンチまでの期間は？

最短即日（数十分）、最大でも 1 営業日以内にローンチします。企画投稿後、AI が即座に解析・スコアリングし、合格ライン超えで完全自動開発が走ります。preview リンクをダッシュボードでお知らせ、あなたが OK で公開ボタンを押すだけ。

プロモーションはどうなりますか？

PR Times プレスリリース配信・Wix ブログ掲載・プエンテ公式 SNS からの紹介投稿はプエンテ広報チームが代行します。オーナー側は、ストアトップの各 SaaS カード右上のシェアアイコンから X / Threads / Facebook / LinkedIn / note / はてブ へワンクリックで投稿可能です。ダッシュボードからは AI 生成の投稿文・OG 画像・縦横動画キットもダウンロードできます。

作ったSaaSのソースコードは貰えますか？

ソースコード・インフラの知的財産権はプエンテに帰属します。ただし、ユーザーのアイデア・コンテンツに関する権利はユーザーに帰属します。
