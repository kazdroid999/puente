# Punete Micro SaaS Store — 事業計画書

最終更新: 2026-04-15
運営: 株式会社プエンテ (PUENTE Inc.)

## 1. サービス概要

「アイデアを 3 日で SaaS に」を掲げる、日本発 Micro SaaS マーケットプレイス。

アイデアオーナーがダッシュボードから企画（概要・対象ユーザー・欲しい機能）を投稿 → Claude が事業計画・BEP・技術スタックを自動生成 → Puente が承認 → Claude Cowork + ボリビアジュニアエンジニアが実装 → **`puente-saas.com/apps/{category}/{slug}` 配下で 3 日以内に公開**。

## 2. バリュープロポジション

| ターゲット | 提供価値 |
|---|---|
| 個人事業主・士業・中小企業 | 自分では作れない SaaS アイデアを低リスクで形にできる |
| 副業・スタートアップ予備軍 | 初期設定費用だけで SaaS オーナーになれる（月額プラットフォーム費 0 円） |
| 既存 SaaS プレイヤー | AEO/SEO 最適化済み・Puente Editorial 掲載で流入獲得 |

## 4. 売上分配（売上分配方式・ロイヤリティではない）

| 受取者 | シェア | 帳簿計上 | インボイス |
|---|---|---|---|
| PUENTE Inc. | **70%** | Puente の売上 | 顧客への発行不要（ユーザーが発行） |
| アイデアオーナー | **30%** | オーナー自身の売上 | オーナーが必要に応じ顧客向け発行 |

- Stripe `application_fee_percent: 70` により自動振分け
- 源泉徴収対象外（ロイヤリティ支払いではないため）
- 翌月末、オーナー指定口座へ Stripe Payout で自動振込

## 5. 技術スタック（固定）

- **Frontend**: Next.js 14 App Router on Cloudflare Pages
- **API**: Cloudflare Workers (Hono) / `api.puente-saas.com`
- **DB**: Supabase Postgres 15 + RLS
- **Auth**: Supabase Auth (Magic Link + Google OIDC)
- **決済**: Stripe Connect (Standard, JP) + Destination Charges
- **AI（設計・コード生成）**: Anthropic Claude Sonnet 4.6 via **Claude Cowork**
- **Email**: Resend
- **SCM / Docs**: GitHub + Notion
- **Observability**: Cloudflare Workers Observability + GA4

## 6. オペレーション体制（超軽量）

- **開発**: 基本すべて **Claude Cowork** が担当。人間はレビューと承認のみ
- **ジュニアエンジニア（ボリビア）1 名**: Cowork の作業チェック・手動修正・顧客対応
- **Puente スーパー管理者**: 承認・PR・マーケ方針決定（兼任・別業務の一部として稼働）
- **プロモーション**: PR Times、Wix Blog、X、Instagram、Facebook、TikTok、YouTube Short を 10 分 Cron で自動配信

人件費を最小化し、Claude Cowork の生産性で回す構造。

### 9.1 Claude API コストのプロキシ化（¥200K → ¥10K に圧縮）

旧計画の月 ¥200,000 → 月 ¥10,000 へ削減する技術スタック：

| 施策 | 効果 | 実装 |
|---|---|---|
| **Cloudflare AI Gateway** | リクエスト一元化・キャッシュ・分析 | `api.puente-saas.com` 経由で Anthropic に Proxy |
| **Prompt Caching** | 共通プロンプト部分（システム・テンプレ）を最大 90% off | Anthropic SDK の `cache_control` 利用 |
| **Result Caching（KV 24h）** | 同一 brief は 24h 間 KV から返却 | Cloudflare KV、key = sha256(brief) |
| **Message Batches API** | 非リアルタイム処理は 50% off | プロモ生成・SNS コピー生成は Batch にキューイング |
| **モデル階層化** | 簡易タスクは Haiku、設計のみ Sonnet | brief 解析=Sonnet、SNS文=Haiku 4.5 |
| **トークン上限制御** | output 上限 + フォーマット強制 | `max_tokens: 1500`、JSON Schema 強制 |

| ワークロード | 件数 / 月 | モデル | input | output | 単価 (cached) | 月額 |

実トークンコストは ¥1,200/月 程度。¥10,000 はバッファ込み保守的見積もり。

## 10.5 マーケティング戦略 — バイラル動画ファースト

広告予算は最小化（¥50,000/月）。獲得チャネルは以下を主軸とする：

### A. Puente 公式チャネルの動画コンテンツ
- **YouTube Short / TikTok / Instagram Reels / Facebook Reels**: 公開された Micro SaaS の紹介ショート動画を自動生成・自動投稿（10 分 Cron）
- **X（旧 Twitter）**: 新規 SaaS ローンチ告知 + 月次 GMV ランキングをスレッド投稿
- 動画テンプレ: 15〜30 秒、UI 録画 + 字幕 + Mellow LoFi BGM、AI 自動生成

### B. **オーナー自発拡散機能（新規追加）**
登録された Micro SaaS を企画オーナー自身が積極的に広められる仕組みを提供：

1. **ワンクリック SNS 投稿**: ダッシュボードに「いまシェア」ボタン
   - X / Threads / Facebook / LinkedIn / Note / はてブ にプリセット文＋OG 画像で投稿
   - LP / カテゴリ別ハッシュタグ自動付与（例: `#MicroSaaS #補助金AI`）
2. **動画キット自動生成**: SaaS 公開時に 9:16 / 1:1 / 16:9 の 3 サイズショート動画＋静止画 OG セットを Cowork が自動生成し、ダッシュボードからダウンロード／直接投稿
3. **アフィリエイト風シェアリンク**: `puente-saas.com/apps/{slug}?ref={owner_id}` で UTM 付き計測。流入経由のサブスク獲得を Owner ダッシュボードに可視化
4. **シェアインセンティブ**: 月間紹介経由サブスク数 Top 10 オーナーに翌月 PR Times 特集枠を提供（追加コスト極小）
5. **埋め込みウィジェット**: ブログ・Note 等に貼れる「料金表＋購読ボタン」iframe コードを発行

### C. PR・記事化
- 公開時に PR Times 自動配信
- Wix Blog（puentework.com）に解説記事クロスポスト
- 月次まとめを Note / Zenn にも自動投稿

### KPI
- バイラル動画 1 本あたり再生 5,000 回以上（YouTube Short / TikTok）
- オーナー経由シェア流入比率 30% 以上を 6 ヶ月で達成
- CAC（顧客獲得コスト）: ¥3,000 以下（初期設定費 ¥66K 〜 ¥330K に対して圧倒的低比率）

## 12. リスクと対策

| リスク | 影響 | 対策 |
|---|---|---|
| 開発遅延（3 日 SLA 未達） | ブランド毀損 | Claude Cowork によるテンプレ化自動生成で 90% 以上自動化 |

| 課金ユーザー獲得の遅れ | GMV シェア減 | 公開時の PR/SNS 自動配信で初速確保、Micro SaaS 本数で量を担保 |
